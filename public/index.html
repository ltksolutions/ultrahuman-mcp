Ochrana osobných údajov (GDPR)

Posledná aktualizácia: apríl 2026

1. Prevádzkovateľ

Prevádzkovateľom tejto aplikácie je vlastník nasadenej inštancie Ultrahuman MCP Server.

2. Aké údaje zbierame

  • Emailová adresa (na účely autentifikácie a identifikácie)
  • Ultrahuman Auth Token (šifrovaný, na účely prístupu k vašim zdravotným dátam)
  • Technické údaje o prihlásení (čas, IP adresa)

3. Účel spracovania

Vaše údaje spracúvame výlučne za účelom poskytovania služby MCP servera — teda sprostredkovania prístupu k vašim zdravotným dátam z Ultrahuman prsteňa pre AI asistenta Claude.

4. Právny základ

Spracovanie prebieha na základe vášho výslovného súhlasu (Článok 6(1)(a) GDPR), ktorý udeľujete pri registrácii.

5. Uchovávanie údajov

Vaše údaje uchovávame po dobu aktívnosti vášho účtu. Po deaktivácii účtu sú údaje vymazané do 30 dní.

6. Vaše práva

Máte právo na prístup k údajom, ich opravu, vymazanie, obmedzenie spracovania, prenositeľnosť a odvolanie súhlasu. Tieto práva môžete uplatniť kontaktovaním správcu systému.

7. Bezpečnosť

Vaše Ultrahuman tokeny sú šifrované pomocou AES-256-GCM. Prihlasovacie linky sú jednorazové a platné 30 minút. Komunikácia prebieha výlučne cez HTTPS.

8. Zdieľanie s tretími stranami

Vaše údaje nezdieľame so žiadnymi tretími stranami okrem služieb nevyhnutných pre chod aplikácie (hosting, odosielanie emailov). Zdravotné dáta z Ultrahuman API sa nikdy neukladajú — iba sa v reálnom čase predávajú do Claude.

Podmienky používania

Posledná aktualizácia: apríl 2026

1. Popis služby

Ultrahuman MCP Server je aplikácia, ktorá umožňuje AI asistentovi Claude pristupovať k vašim zdravotným dátam z Ultrahuman prsteňa prostredníctvom Model Context Protocol (MCP).

2. Požiadavky na používanie

  • Musíte mať aktívny účet na ultrahuman.com
  • Musíte mať platný Ultrahuman API token
  • Emailová adresa zadaná pri registrácii musí zodpovedať vášmu Ultrahuman účtu

3. Zodpovednosť

Služba je poskytovaná „tak ako je" bez akýchkoľvek záruk. Prevádzkovateľ nezodpovedá za presnosť zdravotných dát poskytovaných Ultrahuman API, za výpadky služby, ani za rozhodnutia urobené na základe poskytnutých dát.

4. Bezpečnosť účtu

Ste zodpovední za bezpečnosť svojho Ultrahuman API tokenu a prihlasovacieho emailu. MCP URL je osobná — nezdieľajte ju s tretími osobami.

5. Deaktivácia účtu

Správca systému môže deaktivovať váš účet bez predchádzajúceho upozornenia v prípade porušenia podmienok alebo zneužívania služby.

6. Zmeny podmienok

Prevádzkovateľ si vyhradzuje právo zmeniť tieto podmienky. O zmenách budete informovaní emailom.

2. Ako funguje MCP server

Čo je MCP?

Model Context Protocol (MCP) je otvorený štandard od Anthropic, ktorý umožňuje AI modelom ako Claude pripojiť sa k externým nástrojom a dátovým zdrojom. Fungue ako univerzálny konektor — podobne ako USB-C, ale pre AI.

Čo robí tento MCP server?

Tento server pôsobí ako bezpečný sprostredkovateľ medzi Claude a Ultrahuman Partner API. Keď sa v Claude spýtate na svoje dáta:

1
Claude zavolá váš osobný MCP endpoint (URL vo formáte /api/mcp/[váš-token])
2
Server overí váš token, dešifruje váš Ultrahuman API kľúč a pošle požiadavku na Ultrahuman
3
Ultrahuman API vráti všetky vaše metriky pre požadovaný dátum
4
Server vyfiltruje relevantnú časť odpovede a pošle ju späť do Claude, ktorý vám ju zrozumiteľne interpretuje

Oficiálne Ultrahuman API

Tento MCP server používa oficiálne Ultrahuman Partner API. Dokumentáciu nájdete tu:

  • UltraSignal API Documentation — oficiálna dokumentácia na Notion
  • Prehľad Partner API — na blogu Ultrahuman

API používa jediný endpoint: GET https://partner.ultrahuman.com/api/v1/metrics?email=...&date=YYYY-MM-DD s autorizačným kľúčom v HTTP hlavičke. Vracia kompletné dáta pre daný dátum v jednej odpovedi.

Bezpečnosť

Váš Ultrahuman API token je v databáze uložený šifrovane (AES-256-GCM). Nikto — ani správca systému — ho nevidí v čitateľnej forme. Pri každom volaní sa token dešifruje len v pamäti servera na dobu potrebnú na zavolanie Ultrahuman API.

Vaša MCP URL obsahuje 48-znakový náhodný token. Ktokoľvek, kto by tento URL získal, by mohol čítať vaše Ultrahuman dáta — preto ju nezdieľajte.

3. Ako získať Ultrahuman API token

Ultrahuman Partner API je k dispozícii bezplatne na vyžiadanie. Postup je jednoduchý:

1
Pošlite email na support@ultrahuman.com
Alternatívne môžete použiť aj chat podpory priamo v Ultrahuman mobilnej aplikácii (Profile → Support).
2
V emaile požiadajte o API Authorization Key a Partner ID. Nezabudnite uviesť email, ktorý používate pre Ultrahuman konto — urýchli to spracovanie žiadosti.
3
Po obdržaní odpovede otvorte Ultrahuman mobilnú aplikáciu → záložka Profile → ikona nastavení (ozubené koliesko) vľavo hore → nájdite Partner ID a zadajte ho
4
Zadaný API Authorization Key skopírujte a vložte do poľa „Nový token" v Dashboarde tejto aplikácie

Vzorový email pre Ultrahuman (anglický)

Skopírujte a upravte podľa seba — nahraďte žlté polia svojimi údajmi:

To: support@ultrahuman.com
Subject: Request for Partner API access — personal use
Hi Ultrahuman team, I'd like to request access to the Ultrahuman Partnership API for personal use with my Ultrahuman Ring. I plan to connect my ring data to an AI assistant (Claude via the Model Context Protocol) so I can get personalised insights about my sleep, recovery and HRV. The integration reads only my own data and is not intended for any commercial purpose or redistribution. Could you please provide me with: • API Authorization Key • Partner ID (to enter in the Ultrahuman app under Profile → Settings → Partner ID) My Ultrahuman account email: your-ultrahuman-email@example.com Thank you very much for your help. I'm looking forward to your reply. Best regards, Your Name
Otvoriť v emailovom klientovi
Poznámka k Partner ID Partner ID zadáte priamo v Ultrahuman mobilnej aplikácii (Profile → Settings → Partner ID). Tento krok dáva spoločnosti Ultrahuman súhlas, aby mohla sprístupňovať vaše dáta cez Partner API. Bez neho API volania vrátia chybu aj napriek platnému tokenu.
Ako dlho platí token? Podľa oficiálnej dokumentácie platí access token približne 1 týždeň. V prípade exspirácie alebo problémov s prístupom kontaktujte Ultrahuman podporu znova.

4. Čo obsahuje MCP server

Server sprístupňuje Claude 11 nástrojov (tools). Claude ich volá automaticky na základe vašej otázky — nemusíte ich volať manuálne.

Nástroj Čo vracia
get_all_metrics Kompletné dáta za daný deň — všetky metriky naraz
get_sleep Spánok: skóre, fázy (hlboký/ľahký/REM/prebúdzanie), efektivita, graf HR/HRV počas spánku, AI insights
get_heart_rate Tepová frekvencia: priebeh počas dňa, pokojový tep v noci (night_rhr)
get_hrv Heart Rate Variability — denný priebeh a priemer
get_steps Kroky — priebeh a denný priemer
get_temperature Teplota kože počas spánku v °C
get_recovery Recovery Score (0–10) a Recovery Index (0–100)
get_movement Pohyb a Movement Index (0–100)
get_vo2max Odhad VO2 Max (kardiorespiračná zdatnosť)
get_glucose Glukóza, metabolický skóre, HbA1c, time in target (vyžaduje Ultrahuman M1 CGM)
get_date_range Všetky metriky za rozsah dátumov (max 14 dní) — pre trendy a porovnania
Optimalizácia Server má zabudovaný 60-sekundový cache. Ak sa Claude v rámci jednej otázky pýta na viacero metrík za rovnaký dátum (napr. get_sleep + get_hrv), Ultrahuman API sa zavolá len raz.

5. Skill pre interpretáciu dát

Čo je Claude Skill?

Skill je balík inštrukcií, ktorý Claude načíta, keď sa mu zíde — v tomto prípade pri otázkach o zdravotných dátach. MCP server poskytuje dáta, skill poskytuje know-how, ako ich interpretovať.

Bez skillu Claude interpretuje Ultrahuman dáta na základe všeobecných znalostí. So skillom má konzistentnejšie odporúčania, pozná typické prahy pre HRV/RHR, vie používať workflow šablóny a dodržiava bezpečnostné pravidlá (nediagnostikuje, pri extrémoch odporúča lekára).

Čo obsahuje skill ultrahuman-interpretation?

SKILL.md (hlavný súbor)

  • Kľúčový princíp: osobná baseline prevažuje nad absolútnymi hodnotami
  • Prehľad metrík a ich typov v API odpovedi
  • Bezpečnostné pravidlá (nediagnostikovať, neprepísať lieky, neradiť extrémne intervencie)
  • 5 hlavných workflow šablón: ranný check-in, tréningové rozhodovanie, sleep review, weekly review, detekcia choroby

references/metrics-guide.md

  • Detailné interpretačné rozsahy pre každú metriku
  • Typické hodnoty podľa veku a fitness úrovne
  • VO2 Max tabuľka podľa veku a pohlavia
  • Výklad jednotlivých komponentov v sleep summary

references/safety.md

  • Red-flag vzory (akútna choroba, overtraining, spánková porucha, metabolická dysfunkcia)
  • Zoznam čo povedať a čo nehovoriť
  • Pravidlá pre tehotenstvo a citlivé situácie
  • Ako reagovať, keď dáta protirečia pocitu používateľa

references/workflows.md

  • Rozšírené scenáre: kontrola po alkohole, jetlag, menštruačný cyklus
  • Training load review, pred-súťažný taper
  • Nový používateľ, dlhodobé trendy
  • Interpretácia glukózových dát

Inštalácia skillu do Claude.ai

1
Stiahnite súbor ultrahuman-interpretation.skill.
Stiahnuť skill
2
V Claude.ai kliknite na ikonu profilu → Settings
3
V bočnom menu nájdite Capabilities → Skills (alebo priamo Skills, podľa verzie)
4
Kliknite Upload skill a nahrajte stiahnutý .skill súbor
5
Hotovo. Skill sa aktivuje automaticky, keď sa pýtate na Ultrahuman dáta — nemusíte ho manuálne zapínať

Ako sa skill používa?

Skill funguje v pozadí. Keď napíšete otázku, ktorá spadá do jeho oblasti (spánok, recovery, HRV, tréningové rozhodnutia), Claude:

  1. Zavolá príslušný MCP nástroj (napr. get_sleep)
  2. Načíta skill inštrukcie
  3. Ak je otázka zložitá, načíta aj relevantný reference súbor (napr. metrics-guide.md)
  4. Interpretuje dáta podľa pravidiel skillu a odpovie vám

Výsledok: namiesto generických textov dostanete konkrétne čísla, porovnania s vašou baseline a konkrétne odporúčanie.

Úprava skillu podľa vlastných potrieb

Skill je len obyčajný ZIP archív s Markdown súbormi. Môžete si ho prispôsobiť:

1
Premenujte súbor z .skill na .zip a rozbaľte
2
Upravte obsah podľa potreby — môžete napríklad:
  • Zmeniť tón (viac formálny / uvoľnenejší)
  • Pridať vlastné trénerské inštrukcie do workflows.md
  • Upraviť interpretačné rozsahy v metrics-guide.md podľa vašich špecifík (napr. ak ste extrémny športovec a bežné rozsahy nesedia)
  • Rozšíriť bezpečnostné pravidlá v safety.md
3
Zbalte späť do ZIP archívu — dôležité: adresár ultrahuman-interpretation/ musí byť v koreni ZIP-u
4
Premenujte späť na .skill a nainštalujte podľa krokov vyššie (pôvodnú verziu predtým odstráňte)
Poznámka pre vývojárov Štruktúra skillu musí zachovať YAML frontmatter v SKILL.md (name a description). Popis (description) rozhoduje o tom, kedy sa skill spustí — ak ho zmeníte príliš úzko, skill sa nemusí aktivovať pri relevantných otázkach.

Obmedzenia

  • Skill je potrebné nainštalovať individuálne v každom Claude účte — nedá sa zdieľať cez MCP server
  • Skill beží len v Claude.ai (web, desktop, mobile) — nie v Claude API pokiaľ nie je volaný cez Skills v API
  • Interpretácia je orientačná — nenahrádza lekársku konzultáciu
