Ochrana osobných údajov (GDPR)

Posledná aktualizácia: apríl 2026

1. Prevádzkovateľ

Prevádzkovateľom tejto aplikácie je vlastník nasadenej inštancie Ultrahuman MCP Server.

2. Aké údaje zbierame

  • Emailová adresa (na účely autentifikácie a identifikácie)
  • Ultrahuman Auth Token (šifrovaný, na účely prístupu k vašim zdravotným dátam)
  • Technické údaje o prihlásení (čas, IP adresa)

3. Účel spracovania

Vaše údaje spracúvame výlučne za účelom poskytovania služby MCP servera — teda sprostredkovania prístupu k vašim zdravotným dátam z Ultrahuman prsteňa pre AI asistenta Claude.

4. Právny základ

Spracovanie prebieha na základe vášho výslovného súhlasu (Článok 6(1)(a) GDPR), ktorý udeľujete pri registrácii.

5. Uchovávanie údajov

Vaše údaje uchovávame po dobu aktívnosti vášho účtu. Po deaktivácii účtu sú údaje vymazané do 30 dní.

6. Vaše práva

Máte právo na prístup k údajom, ich opravu, vymazanie, obmedzenie spracovania, prenositeľnosť a odvolanie súhlasu. Tieto práva môžete uplatniť kontaktovaním správcu systému.

7. Bezpečnosť

Vaše Ultrahuman tokeny sú šifrované pomocou AES-256-GCM. Prihlasovacie linky sú jednorazové a platné 30 minút. Komunikácia prebieha výlučne cez HTTPS.

8. Zdieľanie s tretími stranami

Vaše údaje nezdieľame so žiadnymi tretími stranami okrem služieb nevyhnutných pre chod aplikácie (hosting, odosielanie emailov). Zdravotné dáta z Ultrahuman API sa nikdy neukladajú — iba sa v reálnom čase predávajú do Claude.

Podmienky používania

Posledná aktualizácia: apríl 2026

1. Popis služby

Ultrahuman MCP Server je aplikácia, ktorá umožňuje AI asistentovi Claude pristupovať k vašim zdravotným dátam z Ultrahuman prsteňa prostredníctvom Model Context Protocol (MCP).

2. Požiadavky na používanie

  • Musíte mať aktívny účet na ultrahuman.com
  • Musíte mať platný Ultrahuman API token
  • Emailová adresa zadaná pri registrácii musí zodpovedať vášmu Ultrahuman účtu

3. Zodpovednosť

Služba je poskytovaná „tak ako je" bez akýchkoľvek záruk. Prevádzkovateľ nezodpovedá za presnosť zdravotných dát poskytovaných Ultrahuman API, za výpadky služby, ani za rozhodnutia urobené na základe poskytnutých dát.

4. Bezpečnosť účtu

Ste zodpovední za bezpečnosť svojho Ultrahuman API tokenu a prihlasovacieho emailu. MCP URL je osobná — nezdieľajte ju s tretími osobami.

5. Deaktivácia účtu

Správca systému môže deaktivovať váš účet bez predchádzajúceho upozornenia v prípade porušenia podmienok alebo zneužívania služby.

6. Zmeny podmienok

Prevádzkovateľ si vyhradzuje právo zmeniť tieto podmienky. O zmenách budete informovaní emailom.
